This lil' applet is the viewer for Hangman games. HTML, Javascript with JQuery, CSS. The broadcaster is a lil' Ruby code running somewhere - perhaps on your machine. Hangman events are broadcast realtime via Firebase API. So cool.
Also, you can view past games by clicking on the left.

This is my implementation of Project Hangman from the Odin Project.
